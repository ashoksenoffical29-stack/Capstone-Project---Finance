## Capstone-Project---Finance:
This project delivers an end-to-end analytics solution for understanding loan performance, customer behavior, and financial risk in the banking domain. Using real-world KPIs and data models, the project transforms raw loan data into interactive insights for informed decision-making.


# Project Objectives
- Analyze bank loan data to assess portfolio health
- Track loan applications, funded amounts, and repayments
- Differentiate Good Loans vs Bad Loans
- Evaluate financial metrics like interest rate, DTI, and repayment status
- Build dashboards for trend tracking and segmentation
- Enable data-driven lending decisions


## Key Features:

# Data Processing
- Cleaned and transformed dataset using Python and Excel
- KPI calculations implemented through SQL queries
- Segmentation by loan status, purpose, term, employment length, state, grade, DTI, etc.

# Business Insights
- Identified high-risk loan categories using status segmentation
- Highlighted regions with strong/weak loan performance
- Analyzed repayment patterns and borrower stability
- Revealed trends influencing lending decisions

# Jupyter Notebook Analysis
This project includes a complete Python-based data analysis workflow implemented in a Jupyter Notebook.The notebook covers data cleaning, transformation, exploratory analysis, KPI generation, and visualization.

- Notebook File: [https://github.com/ashoksenoffical29-stack/Capstone-Project---Finance/blob/main/Bank%20loan%20analytics.py.ipynb](https://github.com/ashoksenoffical29-stack/Capstone-Project---Finance/blob/main/Bank%20loan%20analytics.py.ipynb)

# Key Steps Performed in Jupyter Notebook
- Imported and cleaned raw bank loan data
- Handled missing values, inconsistencies, and formatting issues
- Conducted exploratory data analysis (EDA)
- Calculated KPIs like loan applications, funded amount, repayment totals, interest rates, and DTI
- Identified Good vs Bad loans using logic-based segmentation
- Performed correlation analysis between loan attributes
- Visualized distributions and trends using Matplotlib/Seaborn
-Exported cleaned dataset for Power BI/Tableau dashboard creation
